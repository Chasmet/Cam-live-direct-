# Cam Live Direct

Application Android pour enregistrer l'ecran du telephone avec le micro et une camera selfie flottante.

## V1

- Interface mobile.
- Capture ecran.
- Camera flottante.
- Bouton demarrer.
- Bouton arreter via notification.
- APK construit automatiquement avec GitHub Actions.

## Installation

Va dans l'onglet Actions de GitHub, ouvre le dernier build, puis telecharge l'artefact APK.
